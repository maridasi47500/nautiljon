Règles à suivre

GÉNÉRAL

  • Considérant qu'une seule version francophone suffit, les DVD ou Blu-ray édition Belge ne sont pas autorisés.
  • Concernant l'image du DVD / Blu-ray, préférez les images en "3D" lorsqu'elles sont disponibles, plutôt que les simples jaquettes.

RÉDACTION & FORMULAIRE

  • Le résumé doit être le résumé du site éditeur, saufs cas exceptionnels ou ce dernier n'est vraiment pas pertinent / n'existe pas.
  • Dans la mesure du possible, respectez au maximum les titres des éditeurs. Pour les cas où aucun titre spécifique n'est donné, la nomenclature à suivre est la suivante :

  • DVD Simple : Titre de l'œuvre - DVD / Exemple : Mon Voisin Totoro - DVD
  • Blu-ray Simple : Titre de l'œuvre - Blu-ray / Exemple : Le Château dans le Ciel - Blu-ray
  • Combo : Titre de l'œuvre - DVD + Blu-ray / Exemple : Arrietty, le Petit Monde des Chapardeurs - Blu-ray + DVD
  • Collector sans coffret : Titre de l'œuvre - Édition Collector - DVD / Exemple : Le Château ambulant - Édition Collector - Format
  • Collector limité : Titre de l'œuvre - Édition Collector Limitée - Format / Exemple : Le Château dans le Ciel - Édition Collector Limitée - Format
  • Collector avec coffret : Titre de l'œuvre - Titre du coffret - Format / Exemple : Arrietty, le Petit Monde des Chapardeurs - Coffret Prestige - Blu-ray + DVD

Dans tous les cas, le format (DVD, Blu-ray, VHS…) doit impérativement apparaître à la fin.

  • La vidéo (bande-annonce) proposée sur une fiche ne doit pas être le trailer de la série / du film, mais du DVD / Blu-ray concerné. Elle doit présenter le support physique en question et non l'œuvre. ATTENTION : seules les vidéos officielles d'éditeurs sont acceptées.

CAS PARTICULIERS

  • Les packs vendus sur Amazon ou d'autres sites comme Anime-Store comprenant plusieurs Box notamment d'animes, ne sont pas acceptés sur le site.
Exemple : Blood+ The Last Vampire - Pack 2 Coffrets, déjà disponible ici : Coffret 1 et Coffret 2

Liens UTILES

  • DVDFr, Amazon, Comme au Cinéma, sites éditeurs
