Règles à suivre

GÉNÉRAL

  • Seuls les produits faisant référence à un artiste ou une œuvre asiatiques sont acceptés.
  • Les annonces sont refusées (autorisées seulement pour les modérateurs). Les prototypes non peints et peints sont acceptés à condition d'avoir une source fiable (ex : lien éditeur ou distributeur).
  • Si vous proposez un produit lié à un personnage ou une personnalité qui porte son nom (ex. : une figurine), préférez le nom au format NOM + PRÉNOM afin de respecter la nomenclature originale.
  • Dans la mesure du possible, il est préférable de respecter au maximum le nom du produit donné par le fabricant. Un format de titre est à préconiser si possible pour les figurines : "Nom de la série - Nom du personnage - Collection (Éditeur)". Exemple : Shingeki no Kyojin - Eren Yeager - ARTFX J (Kotobukiya), ou s'il y a plusieurs version du personnage : "Nom de la série - Nom du personnage - Version du personnage - Collection (Éditeur)".
  • Toujours concernant le titre du goodies proposé, favorisez les noms originaux plutôt qu'internationaux, y compris pour les noms de personnages. Exemple : Fire Emblem : Kakusei - Sallya (Max Factory) en titre alternatif, la version occidentale : Fire Emblem : Awakening - Tharja (Max Factory). Si le goodie est édité par un fabricant français, merci de mettre en premier lieu le titre VF et en titre alternatif le titre original.
  • Les figurines créées par des amateurs ne sont pas acceptées sur le site.
  • Lorsqu'il existe plusieurs versions d'une œuvre, les liaisons avec les œuvres sont à faire en fonction de la figurine ou du goodies : si celle-ci / celui-ci provient exclusivement de la version anime, il est à lier à l'anime uniquement. S'il n'y a aucune précision, il faut lier toutes les versions. Exemple, si un personnage existe en manga et anime, il faut lier le manga, l'anime (sauf OAV/Special), la version manga et la version anime du personnage.
  • Seuls les goodies officiels sont acceptés.
  • Nous acceptons uniquement les posters venant d'éditeurs officiels (Banpresto, Ensky, Animate, etc). Les posters trouvables dans les magazines sont quant à eux refusés.
  • Si la collection n'existe pas, demandez-la lors de la 1ère proposition de goodie. Un modérateur la créera afin que vous proposiez des fiches complètes.


GOODIES MUSIQUE

  • Le format des titres est : « Groupe - Nom+Prénom ou Nom de scène - Nom du goodie (Collection) ». Exemple : EXO-SC - Chan Yeol - Bag Charm (Japan Special Event 2020)
  • Le goodie est toujours lié à au moins une personnalité. Dans le cas d'un goodie d'un membre de groupe, il faut lier uniquement le membre. Exemple : le bag charm de Chan Yeol du groupe EXO-SC ne doit être lié qu'à Chan Yeol, et pas à son groupe EXO-SC.
  • Les goodies similaires de la même collection doivent être liées. Exemple : dans la collection Japan Special Event 2020, les bag charms de chaque membre du groupe EXO-SC, doivent être liés ensemble.
  • Une collection est en lien avec un événement (sortie de CD, tournée, collaboration....), l'ensemble des goodies proposées doit être liées à la collection correspondante (en sélectionnant l'éditeur).
  • Si vous voulez proposer des goodies qui ont plusieurs variantes (ex : EXO-SC - Chan Yeol - Bag Charm (Japan Special Event 2020) et EXO-SC - Se Hun - Bag Charm (Japan Special Event 2020), merci de proposer qu'un goodie. Le modérateur validera votre proposition avec des corrections si nécessaire afin que vous les appliquer à vos fiches suivantes.
  • Un pamphlet doit être proposé en photobook.


AIDE POUR LES MATIÈRES

Il existe des centaines de matières, on ne peut pas toutes les lister dans le détail donc :

  • Pour tous les dérivés de plastiques autres que ceux listés (exemple : POM), il faut renseigner la matière "plastique".
  • Pour la matière PMMA, il faut renseigner Plexiglas / verre acrylique.


CAS PARTICULIERS

  • Si des figurines sont vendues en pack ET séparément, il faut proposer chaque figurine et non pas le pack. Exemple : Nendoroid Petite : Death Note - Case File #01, chaque figurine est venue à l'unité, 500 yens.
  • Les liaisons entre goodies / figurines sont à faire si ceux-ci possèdent une version alternative ou lorsque les figurines appartiennent à une paire ou à un ensemble (des figurines s'imbriquant pour former une scène par exemple). Un ensemble n'est pas une collection !
